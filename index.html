What is a deepfake?
A deepfake refers to AI-generated media where a person‚Äôs appearance or voice is synthetically replaced by another. Although the term emerged around 2017, visual forgery has existed for over a century. VFDNET (Vision Fake Detection Network) is our response ‚Äî a transformer-based deep learning model for spotting fake or GAN-generated images.

üîç How Vision Fake Detection Network (VFDNET) Works

  • Input Image: 224√ó224√ó3 RGB image
  • Patch Embedding: Image split into 16√ó16 patches (196 total), flattened and projected
  • Positional Encoding: Added to patch embeddings to preserve spatial info
  • [CLS] Token: Added to represent global image features
  • Transformer Encoder: Includes Layer Norm, MHSA, MLP, and residual connections
  • Feature Extraction: Use [CLS] token output
  • Classification Head: MLP + softmax for Real/Fake prediction

In an age of synthetic media, VFDNET helps defend visual integrity by detecting manipulated content with high accuracy.

üß™ Ongoing Research


Retinal Disease Detection using Self-Supervised Vision Transformers:
Investigating a scalable AI pipeline combining Self-Supervised Learning (SSL) and Vision Transformers (ViTs) for early detection of retinal diseases such as Diabetic Retinopathy (DR), AMD, and Glaucoma from fundus images. SSL frameworks like DINO and MAE enable the model to learn from large unlabeled datasets, reducing reliance on expert annotations. Fine-tuned ViTs capture global retinal patterns, and Grad-CAM provides visual interpretability. This approach supports telemedicine, clinical triaging, and discovery of novel biomarkers.

An Integrated Deep Learning and Language Processing Framework for Medicinal Plant Identification and Ethnobotanical Description

This project aims to develop an Ethnobotanical Knowledge Graph by integrating textual ethnobotanical knowledge and custom plant datasets through advanced AI and NLP techniques. The workflow is as follows:

  • Text Sources (traditional texts, modern literature, fieldwork) undergo NLP processing for entity recognition, relationship extraction, and standardization.
  • Custom Plant Image Dataset is collected and used for model training via:
    • Traditional ML (SVM, RF, KNN)
    • Deep Learning (CNNs, ResNet, EfficientNet)
    • Transformer-based Models
    • Hybrid and Ensemble Methods
  • Trained models support Plant Identification, followed by Model Comparison and XAI Tools.
  • Outputs from both textual and visual pipelines feed into an Integrated Pipeline for structured knowledge extraction.
  • The resulting knowledge graph informs:
    • Multimodal Reports
    • Web Application
    • Delivery to End Users

Ultimately, this facilitates a comprehensive understanding and preservation of indigenous medicinal plant knowledge by linking plants to their uses, properties, and cultural context.
