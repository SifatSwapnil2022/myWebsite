Capstone Thesis

An Integrated Deep Learning and Language Processing Framework for Medicinal Plant Identification and Ethnobotanical Description

Combining Computer Vision and NLP for biodiversity conservation and traditional medicine research.

  • Dataset: Collected from BAU, Sylhet, Chittagong, and Mymensingh regions.
  • Vision Models: CNN-based classifiers including ResNet50 and EfficientNet, and Transformer-based models.
  • Language Models: T5-small and DistilGPT-2 used for generating ethnobotanical descriptions.
  • Explainability: Integrated Grad-CAM for interpretability of predictions.
  • Deployment: Interactive web app using Streamlit and Flask with user feedback functionality.
  • Supervisor: Dr. Md. Nawab Yousuf Ali
